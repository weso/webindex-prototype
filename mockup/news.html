Visualisation of the Week: Web Index and CIA World Factbook data

15 August, 2014 by Gabe Trodd

  • Latest News
  • News & Media

Our Visualisation of the Week has been produced by Niels Doekemeijer and Jeff Smits. It is an interactive visualisation of selected features from the Web Index 2013 along with selected features from the CIA World Factbook.

Using a deformable world map and a parallel coordinates plot, the visualisation allows users to display various dimensions geographically, by using the input box on the top right. Low values make countries appear smaller, whilst high values make countries appear larger. The low values are coloured red, mid-values are coloured yellow and higher values are coloured green. If there is no data available on a country in any of the selected dimensions, the colour white is displayed in the map.

The parallel coordinates chart uses a minimal alphanumeric format to identify each of the dimensions displayed.

Visualisation of the Week - donuts!

8 August, 2014 by Gabe Trodd

  • News & Media

This week, our visualisation is an interactive tool to allow users to gain a deeper understanding of our Web Index research in the way they choose. It was produced by Damien Robin.

Keeping things simple, Damien presents the Web Index findings with one 'donut' per country, inviting users to order countries by the criteria they want.
